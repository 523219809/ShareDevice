# ShareDevice
轻量级 设备共享 程序 兼容 windows 和mac平台一个轻量级的安卓设备共享程序.

特点:

1.最少的环境依赖

2.即开即用,多平台支持(windows ,mac , linux).

3.支持一人操作,多人观看(直播模式)

使用Websockt + .net core开发,底层使用了 minicap 和 monitouch 开源组件.

Websockt 下行为图像传输,上行为操作数据.
